# 📝 Project: Our Anniversary Website

A simple, static website created to celebrate our special day on **May 5th**. This project is hosted using **GitHub Pages** and serves as a digital scrapbook of our favorite moments.

## ✨ Features
*   **Photo Gallery:** A collection of memories from the past year.
*   **Timeline:** A look back at our biggest milestones.
*   **Responsive Design:** Optimized for viewing on both desktop and mobile devices.
*   **Custom Aesthetic:** Styled with a warm, vintage-inspired theme to match our favorite memories.

## 🚀 Built With
*   **HTML5** - Structure and content.
*   **CSS3** - Custom styling and animations.
*   **JavaScript** - (Optional) Used for a countdown timer or interactive elements.

## 📂 Folder Structure
*   `index.html` — The main landing page.
*   `/css` — Stylesheets for the layout and theme.
*   `/assets` — Images, icons, and personal photos.
